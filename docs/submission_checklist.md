# Submission Checklist

## Required Deliverables
- [ ] Public GitHub repository created
- [ ] Source code pushed to main branch
- [ ] README includes setup and run steps
- [ ] Architecture overview included
- [ ] Greenfield, brownfield, and ambiguous examples included
- [ ] Tests included and passing
- [ ] Repository link shared with hiring team

## Validation Evidence
- [ ] `pytest -q` passes locally
- [ ] `/docs` Swagger page loads
- [ ] URL shortener endpoints verified
- [ ] `/engineering/run` verified for all 3 scenario types

## Final Sanity Checks
- [ ] No secrets committed (`.env`, tokens, credentials)
- [ ] `url_shortener.db` excluded from git
- [ ] README updated with known limitations and trade-offs
- [ ] Commit history is clean and descriptive
